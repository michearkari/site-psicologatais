# Tais Eduarda — Psicóloga | Site

Site institucional da psicóloga **Tais Eduarda** — terapia online.
Site estático (HTML + CSS + JavaScript), sem WordPress, sem build, sem dependências.

## Estrutura

```
psicologa-tais-site/
├── index.html      # página única (CSS e JS embutidos)
├── logo.svg        # logomarca (borboleta + monograma TE) — usada como favicon
└── img/
    ├── tais-1.jpeg # foto do topo (hero)
    ├── tais-2.jpeg # foto da seção "Sobre mim"
    └── tais-3.jpeg # foto da seção "Contato"
```

## Como visualizar

Basta abrir o `index.html` no navegador (duplo-clique).

## Deploy

Por ser estático, pode ser hospedado em qualquer serviço:

- **Hostinger** — subir os arquivos para a pasta `public_html` (via Gerenciador de Arquivos ou FTP).
- **GitHub Pages / Netlify** — apontar para este repositório; o `index.html` é servido automaticamente.
